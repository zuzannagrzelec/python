# Projekty testowania (python)

Proste przyklady skryptow w Pythonie.

## Pliki

- `xd.py` - wypisuje tekst na ekranie
- `dodawanie.py` - program pobiera dwie liczby i wykonuje na nich dzialanie (aktualnie mnozenie)

## Uruchomienie

```powershell
python xd.py
python dodawanie.py
```

## Uwagi

Domyslnie uruchamiaj skrypty w tym samym katalogu, w ktorym znajduja sie pliki.
